# Cost Attribution Example

| Request ID | Session ID | Team | Workflow | Model | Input Tokens | Output Tokens | Tool Usage | Total Cost |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| `req_123` | `sess_456` | `platform` | `remote_review` | `gpt-5.4` | `14200` | `3800` | `2 tool calls` | `$1.87` |

Use attribution at this level so a quota or billing decision can be tied back to a specific request, model, workflow, and team instead of becoming an opaque line item later.
